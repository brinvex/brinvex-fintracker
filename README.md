## Brinvex FinTracker

_Brinvex FinTracker_ is a financial tool focused on investment tracking offering detailed insights into investment portfolios.
It is capable of interacting with various banks and brokers.
Designed as both an application for end-users and a library for integration into other systems,
it provides flexible and scalable solutions for financial tracking.

### Features
#### Financial Statement Parsing
Brinvex FinTracker includes robust Financial Statement Parsing capabilities, 
allowing users to seamlessly import and analyze account statements from 
various financial institutions, including banks and brokers. 
The parsing engine supports multiple formats and automatically 
categorizes transactions, providing a clear and accurate view of investment activity. 
This feature ensures that all relevant data is integrated into the platform, 
enabling detailed tracking and performance analysis across your entire portfolio.  
Currently, _Brinvex FinTracker_ supports statement parsing from:
- Interactive Brokers (_IBKR_)
- Revolut
- Fio Bank and Brokerage (Czech Republic)
- Amundi
